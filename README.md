# Bookmaker bot

Non commercial telegram bot for making bets.

### Run
If you want to run this, then set env variables
```
POSTGRES_HOST
POSTGRES_PASSWORD

TELEGRAM_BOT_TOKEN
TELEGRAM_ADMIN_BOT_TOKEN

INVITE_CODE
TELEGRAM_ADMIN_USERNAME
```

Run server via terminal:
```shell script
gradlew bootRun
```
